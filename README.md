# Análisis de Mercado y Ventas de Videojuegos 🎮📈

Este proyecto realiza un **Análisis Exploratorio de Datos (EDA)** y un estudio analítico sobre el mercado global de los videojuegos. El objetivo principal es identificar patrones de éxito comercial, analizar el rendimiento de las diferentes plataformas (consolas), evaluar el impacto de los géneros a lo largo del tiempo y entender el comportamiento de los consumidores en las distintas regiones geográficas (Norteamérica, Europa, Japón, etc.).

---

## 🎯 Preguntas de Negocio Respondidas
A través de este análisis, se da respuesta a las siguientes interrogantes del sector:
* ¿Cuáles son las plataformas que han dominado históricamente el mercado y cuáles muestran una tendencia al declive?
* ¿Cómo varían las preferencias de géneros de videojuegos según la región geográfica?
* ¿Existe una correlación directa entre las puntuaciones de las críticas (scores) y el éxito en ventas globales?
* ¿Cuáles son las distribuidoras (*publishers*) con el mayor catálogo de éxitos comerciales?

---

## 📊 Metodología y Herramientas
El análisis se desarrolló de manera íntegra en Python, utilizando el ecosistema estándar de Ciencia de Datos:

1. **Limpieza de Datos:** Tratamiento de valores nulos, eliminación de duplicados y conversión de tipos de datos (fechas, categorías).
2. **Análisis Estadístico Descriptivo:** Distribución de las ventas globales y regionales.
3. **Visualización de Datos:** Implementación de gráficos avanzados con **Matplotlib** y **Seaborn** (gráficos de líneas para tendencias temporales, *boxplots* para dispersión de ventas por plataforma y *heatmaps* de correlación).
